# resgu-cc

### Install dependencies
```
yarn install
```

### Run Tests
```
yarn test
```

### Part 1 - Simply run tests:
```
yarn test
```

### Part 2 - Run CLI app:
```
node resgu.js
```

to talk resgunian: `talk -t "My name is Okota"`
